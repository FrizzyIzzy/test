CSS

CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts. It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers. CSS is independent of HTML and can be used with any XML-based markup language. It makes things look nice

How To Use CSS

It all starts with using the <style> tag inside the <head> tags.

<head>

<style>

</style>

</head>

With it you are able to….

  1. Color Text
  2. Text Color Background
  3. Align Text and Images
  4. Add Text Decoration

CSS Format


Whenever you use CSS to style your website, you will need to go between the <style> tag and use one of the selectors; id, class, or attribute selector. Alright now that you have selected your desired text you want to style, you will need to write the selector or declaration of choice, add a space and a pair of curly brackets { }. Using round or square brackets will not work in this scenario. The section inside the brackets will vary between how you want to style.
“Selector or Declaration” { }

Example:
h1 {}



Coloring Text


Alright, now that you know the general format to use, you will need the specific aspects inside the curly brackets. To color text you will need to start off with writing “color” followed by “:”. After you have these, you will need to add a space and write “rgb” followed by circular brackets “( )”. In the circular brackets you need to put three numbers that correspond with the rgb value that will be displayed.Then end it off with a “;” before the last curly bracket.

“Selector or Declaration” {color: rgb(x, x, x );}

Example.
#example-one {color: rgb(9, 0, 255);}

Background Text Color


Text color background is very similar to coloring text. Start off with using the general format, then in the curly brackets start off with writing “background-color”, followed by “: rgb(x, x, x);” for your desired color and your closing curly bracket.

“Selector or Declaration” {background-color: rgb(x, x, x );}

Example.
#example-two {background-color: rgb(255, 242, 255);}

This is what background text color looks like





Aligning Text


To align text, you will need to follow the basic CSS format. Inside of the curly brackets you will need to write “text-align:” followed by a space then either “left”, “right” or “center”.

“Selector or Declaration” {text-align: X;}

Example.
#example-three {text-align: center;}

This is what center text align looks like

Add Text Decoration


With CSS we can add other types of text decoration. To do this follow the basic CSS format, inside of the curly brackets start off with “text-decoration:” then followed a space then by either “underline”, “line-through” or “overline”.

“Selector or Declaration” {text-decoration: X;}

Examples.


#example-four {text-decoration: underline;}

Underline




#example-five {text-decoration: line-through;}

Striked through



#example-six {text-decoration: overline;}

Over line
